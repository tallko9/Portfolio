# Portfolio de Sasha Lorenc

## Présentation
Ce dépôt contient mon portfolio personnel présentant mes compétences, projets et expériences. Le site est conçu pour mettre en valeur mon parcours professionnel et académique dans le domaine du développement.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Docker
- CI/CD avec Drone CI

## Structure du projet
- `/css` - Feuilles de style CSS
- `/js` - Scripts JavaScript
- `/images` - Ressources graphiques
- `/html` - Pages HTML additionnelles
- `/documents` - Fichiers téléchargeables (CV, etc.)
- `/docker` - Configuration Docker
